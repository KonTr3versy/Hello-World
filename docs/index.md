# Offensive Security Cheatsheets

A public, authorized-use reference for common red team and penetration testing tools. Content is intentionally concise: quick descriptions, safe usage anchors, and typical engagement scenarios.

## Content standards

- Authorized-use focus only.
- No exploit chains, payloads, or bypass techniques.
- One-sentence tool summaries with minimal command anchors.

# OWASP ZAP

**Category:** Web

**Official Docs:** https://www.zaproxy.org/

**Description:** Intercepting proxy and web assessment suite for authorized web testing.

**Common authorized scenarios:**
- Capturing HTTP traffic for application review.
- Running authenticated passive scans during a test window.
- Documenting findings with repeatable scanner outputs.

**Minimal usage examples:**
```bash
zap.sh -quickurl https://target.example -quickout zap-report.html
```

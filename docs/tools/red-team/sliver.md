# Sliver

**Category:** Red Team

**Official Docs:** https://github.com/BishopFox/sliver

**Description:** Open-source C2 framework for authorized red team operations.

**Common authorized scenarios:**
- Coordinating operator activity during a time-boxed engagement.
- Managing implant lifecycle in a controlled lab or exercise.
- Demonstrating command-and-control readiness with explicit approval.

**Minimal usage examples:**
```bash
sliver-server
```

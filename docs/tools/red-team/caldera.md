# MITRE Caldera

**Category:** Red Team

**Official Docs:** https://github.com/mitre/caldera

**Description:** Automates adversary emulation plans for authorized security testing.

**Common authorized scenarios:**
- Running controlled emulation plans aligned to approved objectives.
- Measuring detection coverage for a specific technique set.
- Coordinating repeatable assessments across environments.

**Minimal usage examples:**
```bash
python3 server.py --insecure
```

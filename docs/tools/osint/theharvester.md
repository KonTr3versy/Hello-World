# theHarvester

**Category:** OSINT

**Official Docs:** https://github.com/laramies/theHarvester

**Description:** Collects emails, subdomains, and related metadata from public sources.

**Common authorized scenarios:**
- Building a public-facing asset list for scoping.
- Verifying exposed contact data for reporting.
- Identifying public subdomains prior to validation.

**Minimal usage examples:**
```bash
theHarvester -d example.com -b all
```

# Maltego

**Category:** OSINT

**Official Docs:** https://www.maltego.com/

**Description:** Graph-based link analysis for open-source intelligence investigations.

**Common authorized scenarios:**
- Mapping public-facing relationships for a scoped target.
- Enriching known entities to support attack surface reviews.
- Producing relationship visuals for reporting.

**Minimal usage examples:**
```bash
maltego
```

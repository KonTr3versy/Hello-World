# Prowler

**Category:** Cloud

**Official Docs:** https://github.com/prowler-cloud/prowler

**Description:** Runs AWS security posture checks and produces concise compliance reports.

**Common authorized scenarios:**
- Capturing an AWS account security baseline for a quarterly review.
- Identifying risky configurations before a red team exercise.
- Generating evidence for remediation tracking.

**Minimal usage examples:**
```bash
prowler aws --checks s3 public-access
```

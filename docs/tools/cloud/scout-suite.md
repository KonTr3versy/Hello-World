# Scout Suite

**Category:** Cloud

**Official Docs:** https://github.com/nccgroup/ScoutSuite

**Description:** Audits cloud environments to highlight configuration risks across providers.

**Common authorized scenarios:**
- Building a multi-account inventory for a scoping exercise.
- Validating security posture before a red team engagement.
- Producing a point-in-time snapshot for reporting.

**Minimal usage examples:**
```bash
scout aws --report-dir ./reports/scout
```

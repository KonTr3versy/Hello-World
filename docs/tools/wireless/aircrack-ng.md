# Aircrack-ng

**Category:** Wireless

**Official Docs:** https://www.aircrack-ng.org/

**Description:** Wireless monitoring and assessment toolkit for authorized testing.

**Common authorized scenarios:**
- Capturing wireless traffic during an approved assessment.
- Validating wireless coverage and monitoring rules.
- Documenting observed SSIDs and encryption types.

**Minimal usage examples:**
```bash
airmon-ng start wlan0
```

# Kismet

**Category:** Wireless

**Official Docs:** https://www.kismetwireless.net/

**Description:** Wireless detection and monitoring platform for situational awareness.

**Common authorized scenarios:**
- Passive discovery of wireless networks in a controlled survey.
- Building an inventory of SSIDs and device types.
- Recording wireless telemetry for reporting.

**Minimal usage examples:**
```bash
kismet -c wlan0
```
